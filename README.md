# VK C# Бот — Мастер-класс

Добро пожаловать на мастер-класс по созданию бота для ВКонтакте на языке C#!

В этом репозитории представлены две версии бота:
- **'echo-bot/'** — простейший эхо-бот (отвечает тем же текстом)
- **'full-bot/'** — полноценный бот с набором команд (!помощь, !время, !калькулятор и др.)

## 📋 Требования перед началом

- [Установите .NET SDK](https://dotnet.microsoft.com/download) (версия 6.0 или новее)
- [Установите Visual Studio 2022](https://visualstudio.microsoft.com/ru/vs/) или [VS Code](https://code.visualstudio.com/)
- Аккаунт ВКонтакте

## 🚀 Пошаговая инструкция

### Шаг 1. Создание сообщества ВКонтакте

1. Перейдите на [vk.com](https://vk.com) и войдите в свой аккаунт
2. Нажмите на **«Создать сообщество»** в правом меню
3. Выберите тип **«Управление бизнесом»** или **«Тематическое»** (любой подойдёт)
4. Придумайте название, например: `Мой первый бот`
5. Нажмите **«Создать сообщество»**

### Шаг 2. Получение API ключа (токена) и настройка Long Poll

1. В управлении сообществом перейдите в раздел **«Управление»** → **«Работа с API»**

2. **Включите Long Poll API** (это важно!)
   - Опуститесь вниз страницы до блока **«Long Poll API»**
   - Поставьте галочку **«Включён»**
   - В разделе **«Версия»** выберите актуальную версию (например, 5.199)
   - В разделе **«События»** убедитесь, что отмечено событие **«Входящее сообщение»**
   - Нажмите **«Сохранить»**

3. Теперь создайте ключ доступа:
   - Нажмите кнопку **«Создать ключ»**
   - В появившемся окне выберите все разрешения:
     - ✅ Разрешение на доступ к сообщениям сообщества
     - ✅ Разрешение на доступ к фотографиям (опционально)
   - Нажмите **«Создать»**

> ⚠️ **Важно!** Long Poll API должен быть включён до создания ключа. Если вы уже создали ключ, а потом включили Long Poll — лучше создать новый ключ.

4. **ВАЖНО!** Скопируйте полученный ключ (токен) и сохраните его в надёжном месте. Токен показывается только один раз!

> 🔑 Пример токена: `vk1.a.tms46HfJsj-6aaOs9UYdcYL-jws29B...`

### Шаг 3. Получение ID сообщества

ID вашей группы можно найти:
- В адресной строке браузера: `https://vk.com/clubXXXXXXXXX`
- Где `XXXXXXXXX` — это и есть ID вашей группы
- Например, для адреса `club238068930` ID будет `238068930`

### Шаг 4. Включение сообщений сообщества

1. В управлении сообществом перейдите в **«Управление»** → **«Сообщения»**
2. Включите опцию **«Сообщения сообщества»**
3. Рекомендуем также включить **«Возможности ботов»** (для кнопок и клавиатур)

### Шаг 5. Создание проекта

#### Вариант А: Через командную строку

```bash
# Создаём папку для проекта
mkdir MyVkBot
cd MyVkBot

# Создаём консольное приложение
dotnet new console

# Устанавливаем библиотеку VkNet
dotnet add package VkNet
```

#### Вариант Б: Через Visual Studio

1. Откройте Visual Studio
2. Нажмите **«Создать новый проект»**
3. Выберите **«Консольное приложение»** (C#)
4. Укажите имя проекта (например, `MyVkBot`)
5. После создания проекта установите через NuGet пакет `VkNet`

### Шаг 6. Запуск эхо-бота

1. Скопируйте код из файла `echo-bot/StartUp.cs` в ваш `Program.cs`
2. Замените строку с токеном на свой:
   ```csharp
   api.Authorize(new ApiAuthParams { AccessToken = "ВАШ_ТОКЕН_СЮДА" });
   ```
3. Замените `groupId` на ID вашего сообщества:
   ```csharp
   ulong groupId = 123456789; // Вставьте ваш ID
   ```
4. Запустите проект:
   - `F5` в Visual Studio
   - или `dotnet run` в командной строке
5. Напишите любое сообщение в ваше сообщество ВКонтакте

### Шаг 7. Запуск полной версии бота

1. Скопируйте в проект три файла из папки `full-bot/`:
   - `StartUp.cs`
   - `MessageHandler.cs`
   - `ConfigModel.cs`
2. Создайте файл `config.json` рядом с `StartUp.cs`:
   ```json
   {
       "AccessToken": "ВАШ_ТОКЕН_СЮДА",
       "GroupId": 123456789
   }
   ```
3. Запустите проект

## 🎮 Доступные команды (полная версия)

## ❓ Частые ошибки и их решение

| Ошибка | Решение |
|--------|---------|
| `Access token is invalid` | Токен недействителен. Получите новый токен в разделе «Работа с API» |
| `Failed to authorize` | Проверьте, что вы скопировали токен полностью (он очень длинный) |
| `Group ID not found` | Убедитесь, что ID группы указан правильно (без `club`) |
| Сообщения не приходят | 1. Включите «Сообщения сообщества» в настройках группы<br>2. Включите Long Poll API в разделе «Работа с API»<br>3. Проверьте, что в событиях Long Poll отмечено «Входящее сообщение» |
| Long Poll не работает | Убедитесь, что вы включили Long Poll API ДО создания ключа. Если нет — создайте новый ключ |
| Не вижу смайликов в консоли | В коде уже есть `Console.OutputEncoding = Encoding.UTF8;`. Если не помогло — запустите консоль от имени администратора |

## 📝 Домашнее задание (для самых любознательных)

Попробуйте добавить свою команду в бота:

1. В файле `MessageHandler.cs` найдите `switch` по командам
2. Добавьте новый `case`:
   ```csharp
   case "!факт":
       string[] facts = { "В C# 9.0 появились записи", "Байт — 8 бит", "VK был создан в 2006 году" };
       Random rnd = new Random();
       SendMessage(msg, facts[rnd.Next(facts.Length)]);
       break;
   ```
3. Запустите бота и напишите ему `!факт`
